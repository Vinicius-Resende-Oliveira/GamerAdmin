O projeto consiste em criar um sistema para jogador receberem pontos por missões concluidas em um determinado jogo;
Neste projeto o usuário enviará para o sistema o seu nome, NIKNAME, Nome do grupo e a foto para a analise do administardor;
Os dados deverão ser guardados no banco de dados;
O adminstrador após receber os dados do usuários visualizará na sua página rezervada os dados dos jogadores com a imagem para ser baixada.
O administrador poderá atribuir pontos para cada missão concluida;
